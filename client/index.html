Introduction

Welcome to the LO State Machine modeling page. This page is dedicated towards communicating the high-level technical design of the LO State Machine and relevant information for aspects of this state machine. The scope/audience of the document/page is structured to focus on the consumer facing aspects of the state machine (and "data on the outside"), rather than the internals (data architecture, implementation of services).
